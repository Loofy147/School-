# System Scope

Defining the scope of a system is a critical first step in the architecture process. It involves drawing a boundary around the system to identify what is inside the system, what is outside, and how the system interacts with its environment.

A well-defined scope prevents "scope creep," where the project's requirements continuously expand, leading to delays and budget overruns.

# نطاق النظام

يعد تحديد نطاق النظام خطوة أولى حاسمة في عملية الهندسة المعمارية. وهو ينطوي على رسم حدود حول النظام لتحديد ما هو داخل النظام، وما هو خارجه، وكيف يتفاعل النظام مع بيئته.

يمنع النطاق المحدد جيدًا "زحف النطاق"، حيث تتوسع متطلبات المشروع باستمرار، مما يؤدي إلى تأخير وتجاوز الميزانية.
