# Tools for AI Strategy & Integration
# أدوات لاستراتيجية الذكاء الاصطناعي والتكامل

## AI & ML Frameworks / أطر الذكاء الاصطناعي وتعلم الآلة

- **TensorFlow / PyTorch:** The two most popular open-source libraries for building and training deep learning models.
- **Scikit-learn:** A widely used Python library for traditional machine learning algorithms (e.g., regression, classification, clustering).
- **Hugging Face Transformers:** A library that provides pre-trained models and tools for a wide range of natural language processing (NLP) tasks.

- **TensorFlow / PyTorch:** أشهر مكتبتين مفتوحتي المصدر لبناء وتدريب نماذج التعلم العميق.
- **Scikit-learn:** مكتبة Python مستخدمة على نطاق واسع لخوارزميات التعلم الآلي التقليدية (مثل الانحدار والتصنيف والتجميع).
- **Hugging Face Transformers:** مكتبة توفر نماذج وأدوات مدربة مسبقًا لمجموعة واسعة من مهام معالجة اللغة الطبيعية (NLP).

## MLOps Platforms / منصات MLOps

- **MLflow:** An open-source platform for managing the end-to-end machine learning lifecycle, including experiment tracking, model packaging, and deployment.
- **Kubeflow:** An open-source toolkit for making deployments of machine learning workflows on Kubernetes simple, portable, and scalable.
- **Vertex AI (Google Cloud) / SageMaker (AWS):** Managed cloud platforms that provide a comprehensive suite of tools for building, training, and deploying ML models.

- **MLflow:** منصة مفتوحة المصدر لإدارة دورة حياة التعلم الآلي من البداية إلى النهاية، بما في ذلك تتبع التجارب وتعبئة النماذج والنشر.
- **Kubeflow:** مجموعة أدوات مفتوحة المصدر لجعل عمليات نشر مهام سير عمل التعلم الآلي على Kubernetes بسيطة ومحمولة وقابلة للتطوير.
- **Vertex AI (Google Cloud) / SageMaker (AWS):** منصات سحابية مُدارة توفر مجموعة شاملة من الأدوات لبناء نماذج تعلم الآلة وتدريبها ونشرها.

## Vector Databases (for RAG) / قواعد بيانات المتجهات (لـ RAG)

- **Pinecone:** A managed, cloud-native vector database.
- **Chroma / Faiss:** Open-source libraries for efficient similarity search and clustering of dense vectors.

- **Pinecone:** قاعدة بيانات متجهة مُدارة ومصممة للسحابة.
- **Chroma / Faiss:** مكتبات مفتوحة المصدر للبحث الفعال عن التشابه وتجميع المتجهات الكثيفة.
