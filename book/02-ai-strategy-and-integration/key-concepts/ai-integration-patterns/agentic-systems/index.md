# Agentic Systems
# الأنظمة الوكيلية

This pattern involves creating AI "agents" that can take autonomous actions to achieve a goal. These agents can use tools (like APIs), reason about multi-step tasks, and even collaborate with other agents. This is a more advanced pattern for complex, goal-oriented automation.

يتضمن هذا النمط إنشاء "وكلاء" ذكاء اصطناعي يمكنهم اتخاذ إجراءات مستقلة لتحقيق هدف ما. يمكن لهؤلاء الوكلاء استخدام الأدوات (مثل واجهات برمجة التطبيقات)، والتفكير في المهام متعددة الخطوات، وحتى التعاون مع وكلاء آخرين. هذا نمط أكثر تقدمًا للأتمتة المعقدة الموجهة نحو الهدف.
