# Tools for Product/UX System Design
# أدوات تصميم المنتج وتجربة المستخدم على مستوى النظام

- **Miro / Mural:** Online collaborative whiteboarding platforms that are excellent for creating user journey maps, service blueprints, and other diagrams.
- **Figma / Sketch:** Vector graphics editors used for creating high-fidelity mockups, prototypes, and UI designs.
- **Jira / Trello:** Project management tools used for tracking user stories, tasks, and bugs, and for managing the development lifecycle.
- **Gherkin:** A simple, structured language used for writing acceptance criteria and end-to-end scenarios.
- **Storybook:** An open-source tool for building UI components and pages in isolation. It streamlines UI development, testing, and documentation.

- **Miro / Mural:** منصات ألواح بيضاء تعاونية عبر الإنترنت ممتازة لإنشاء خرائط رحلة المستخدم ومخططات الخدمة والرسوم البيانية الأخرى.
- **Figma / Sketch:** محررات رسومات متجهة تُستخدم لإنشاء نماذج بالأحجام الطبيعية ونماذج أولية وتصميمات واجهة مستخدم عالية الدقة.
- **Jira / Trello:** أدوات إدارة المشاريع المستخدمة لتتبع قصص المستخدم والمهام والأخطاء وإدارة دورة حياة التطوير.
- **Gherkin:** لغة بسيطة ومنظمة تستخدم لكتابة معايير القبول والسيناريوهات الشاملة.
- **Storybook:** أداة مفتوحة المصدر لبناء مكونات وصفحات واجهة المستخدم بمعزل عن بعضها البعض. إنه يبسط تطوير واجهة المستخدم واختبارها وتوثيقها.
