# Acceptance Criteria
# معايير القبول

A formal list of requirements that must be met for a user story to be considered complete. They are often written in a Gherkin format (Given/When/Then).

---

قائمة رسمية بالمتطلبات التي يجب الوفاء بها حتى تكتمل قصة المستخدم. غالبًا ما تتم كتابتها بتنسيق Gherkin (بالنظر إلى / متى / ثم).
