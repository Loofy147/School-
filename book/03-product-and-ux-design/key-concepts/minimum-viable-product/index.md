# Minimum Viable Product (MVP)
# الحد الأدنى من المنتج القابل للتطبيق (MVP)

Defining the smallest set of features that delivers value to the end-user and can be shipped. This allows for iterative development and learning from user feedback.

---

تحديد أصغر مجموعة من الميزات التي تقدم قيمة للمستخدم النهائي ويمكن شحنها. يسمح هذا بالتطوير التكراري والتعلم من ملاحظات المستخدم.
