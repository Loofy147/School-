# Personally Identifiable Information (PII)
# المعلومات الشخصية المحددة للهوية (PII)

Any data that could be used to identify a specific individual. This includes names, email addresses, phone numbers, and even IP addresses. Protecting PII is a primary goal of this layer.

---

أي بيانات يمكن استخدامها لتحديد فرد معين. يتضمن ذلك الأسماء وعناوين البريد الإلكتروني وأرقام الهواتف وحتى عناوين IP. تعد حماية المعلومات الشخصية المحددة للهوية هدفًا أساسيًا لهذه الطبقة.
