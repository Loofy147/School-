# Defense in Depth
# الدفاع في العمق

The principle of using multiple, layered security controls. If one layer fails, another is there to stop the attack. For example, using a firewall, network segmentation, and application-level authentication together.

---

مبدأ استخدام ضوابط أمنية متعددة ومتدرجة. إذا فشلت طبقة واحدة، فهناك طبقة أخرى لإيقاف الهجوم. على سبيل المثال، استخدام جدار حماية وتقسيم الشبكة ومصادقة على مستوى التطبيق معًا.
