# Encryption
# التشفير

Encryption is the process of converting data into a code to prevent unauthorized access. It is a fundamental security control that should be applied in two primary contexts:

- **[Encryption in-transit](./in-transit/index.md):** Protecting data as it moves between systems.
- **[Encryption at-rest](./at-rest/index.md):** Protecting data when it is stored.

---

التشفير هو عملية تحويل البيانات إلى رمز لمنع الوصول غير المصرح به. وهو عنصر تحكم أمني أساسي يجب تطبيقه في سياقين أساسيين:

- **[التشفير أثناء النقل](./in-transit/index.md):** حماية البيانات أثناء انتقالها بين الأنظمة.
- **[التشفير في حالة السكون](./at-rest/index.md):** حماية البيانات عند تخزينها.
