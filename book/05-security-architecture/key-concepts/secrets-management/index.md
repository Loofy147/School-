# Secrets Management
# إدارة الأسرار

The secure storage, distribution, and rotation of "secrets" like API keys, database passwords, and private certificates. Secrets should never be hard-coded in the source code.

---

التخزين الآمن وتوزيع وتدوير "الأسرار" مثل مفاتيح API وكلمات مرور قاعدة البيانات والشهادات الخاصة. لا ينبغي أبدًا ترميز الأسرار بشكل ثابت في الكود المصدري.
