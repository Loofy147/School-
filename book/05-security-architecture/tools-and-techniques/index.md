# Tools for Security Architecture & Threat Modeling
# أدوات لهندسة الأمن ونمذجة التهديدات

- **Threat Modeling Tools (e.g., OWASP Threat Dragon, Microsoft Threat Modeling Tool):** Applications that help you create threat models by drawing data flow diagrams and identifying potential threats using frameworks like STRIDE.
- **Secrets Management Systems (e.g., HashiCorp Vault, AWS Secrets Manager):** Tools that provide a centralized and secure way to store and manage API keys, passwords, and other secrets.
- **Web Application Firewalls (WAFs):** A type of firewall that operates at the application layer to filter and block malicious HTTP traffic, protecting against common web exploits like SQL injection and cross-site scripting (XSS).
- **Static Application Security Testing (SAST) Tools (e.g., SonarQube, Veracode):** Tools that scan your source code for potential security vulnerabilities without running the code.
- **Dynamic Application Security Testing (DAST) Tools (e.g., OWASP ZAP, Burp Suite):** Tools that test your running application for vulnerabilities by simulating attacks and observing the responses.
- **IAM Platforms (e.g., Okta, Auth0, Keycloak):** Services that provide robust identity and access management, often including features like multi-factor authentication (MFA) and single sign-on (SSO).

- **أدوات نمذجة التهديدات (على سبيل المثال، OWASP Threat Dragon ، Microsoft Threat Modeling Tool):** تطبيقات تساعدك على إنشاء نماذج تهديد عن طريق رسم مخططات تدفق البيانات وتحديد التهديدات المحتملة باستخدام أطر عمل مثل STRIDE.
- **أنظمة إدارة الأسرار (على سبيل المثال، HashiCorp Vault ، AWS Secrets Manager):** أدوات توفر طريقة مركزية وآمنة لتخزين وإدارة مفاتيح API وكلمات المرور والأسرار الأخرى.
- **جدران حماية تطبيقات الويب (WAFs):** نوع من جدران الحماية يعمل على طبقة التطبيق لتصفية وحظر حركة مرور HTTP الخبيثة، مما يحمي من مآثر الويب الشائعة مثل حقن SQL والبرمجة النصية عبر المواقع (XSS).
- **أدوات اختبار أمان التطبيقات الثابتة (SAST) (على سبيل المثال، SonarQube ، Veracode):** أدوات تفحص الكود المصدري بحثًا عن نقاط الضعف الأمنية المحتملة دون تشغيل الكود.
- **أدوات اختبار أمان التطبيقات الديناميكية (DAST) (على سبيل المثال، OWASP ZAP ، Burp Suite):** أدوات تختبر تطبيقك قيد التشغيل بحثًا عن نقاط الضعف عن طريق محاكاة الهجمات ومراقبة الاستجابات.
- **منصات IAM (على سبيل المثال، Okta ، Auth0 ، Keycloak):** خدمات توفر إدارة هوية ووصول قوية، وغالبًا ما تتضمن ميزات مثل المصادقة متعددة العوامل (MFA) وتسجيل الدخول الموحد (SSO).
