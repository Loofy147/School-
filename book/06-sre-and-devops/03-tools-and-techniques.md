# Tools for SRE / DevOps / Observability
# أدوات لـ SRE / DevOps / المراقبة

- **CI/CD Platforms (e.g., Jenkins, GitLab CI, GitHub Actions):** Tools that automate the build, test, and deployment pipelines.
- **IaC Tools (e.g., Terraform, Ansible, AWS CloudFormation):** Tools that allow you to define and manage your infrastructure as code.
- **Containerization & Orchestration (e.g., Docker, Kubernetes):**
    - **Docker:** A platform for creating, deploying, and running applications in lightweight, portable containers.
    - **Kubernetes (K8s):** An open-source system for automating the deployment, scaling, and management of containerized applications.
- **Observability Platforms (e.g., Prometheus, Grafana, Datadog, Jaeger):**
    - **Prometheus:** An open-source monitoring system for collecting and storing metrics.
    - **Grafana:** An open-source platform for visualizing and analyzing metrics, often used with Prometheus.
    - **Jaeger:** An open-source, end-to-end distributed tracing system.
- **Log Aggregation Tools (e.g., ELK Stack - Elasticsearch, Logstash, Kibana; Splunk):** Tools that collect, centralize, and analyze log data from all services in the system.
- **Configuration Management (e.g., Ansible, Puppet, Chef):** Tools that automate the process of configuring and maintaining the state of servers.

- **منصات CI / CD (على سبيل المثال، Jenkins ، GitLab CI ، GitHub Actions):** أدوات تعمل على أتمتة خطوط أنابيب البناء والاختبار والنشر.
- **أدوات IaC (على سبيل المثال، Terraform ، Ansible ، AWS CloudFormation):** أدوات تتيح لك تحديد وإدارة البنية التحتية الخاصة بك ككود.
- **الحاويات والتنسيق (على سبيل المثال، Docker ، Kubernetes):**
    - **Docker:** منصة لإنشاء ونشر وتشغيل التطبيقات في حاويات خفيفة الوزن ومحمولة.
    - **Kubernetes (K8s):** نظام مفتوح المصدر لأتمتة نشر وتوسيع نطاق وإدارة التطبيقات المعبأة في حاويات.
- **منصات المراقبة (على سبيل المثال، Prometheus ، Grafana ، Datadog ، Jaeger):**
    - **Prometheus:** نظام مراقبة مفتوح المصدر لجمع وتخزين المقاييس.
    - **Grafana:** منصة مفتوحة المصدر لتصور وتحليل المقاييس، وغالبًا ما تستخدم مع Prometheus.
    - **Jaeger:** نظام تتبع موزع مفتوح المصدر وشامل.
- **أدوات تجميع السجلات (على سبيل المثال، ELK Stack - Elasticsearch ، Logstash ، Kibana ؛ Splunk):** أدوات تجمع بيانات السجل من جميع الخدمات في النظام وتركزها وتحللها.
- **إدارة التكوين (على سبيل المثال، Ansible ، Puppet ، Chef):** أدوات تعمل على أتمتة عملية تكوين وصيانة حالة الخوادم.
