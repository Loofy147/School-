# Continuous Integration (CI)
# التكامل المستمر (CI)

The practice of frequently merging all developers' code changes into a central repository, after which automated builds and tests are run.

---

ممارسة دمج تغييرات كود جميع المطورين بشكل متكرر في مستودع مركزي، وبعد ذلك يتم تشغيل عمليات البناء والاختبارات التلقائية.
