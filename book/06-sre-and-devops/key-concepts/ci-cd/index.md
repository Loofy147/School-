# CI/CD (التكامل المستمر / النشر المستمر)

CI/CD is the backbone of modern DevOps, creating an automated workflow that allows teams to deliver software reliably and efficiently.

- **[Continuous Integration (CI)](./continuous-integration/index.md):** The practice of frequently merging and automatically testing code changes.
- **[Continuous Deployment (CD)](./continuous-deployment/index.md):** The practice of automatically deploying successful changes to production.

---

CI/CD هو العمود الفقري لـ DevOps الحديثة، حيث ينشئ سير عمل آليًا يسمح للفرق بتقديم البرامج بشكل موثوق وفعال.

- **[التكامل المستمر (CI)](./continuous-integration/index.md):** ممارسة دمج تغييرات الكود واختبارها تلقائيًا بشكل متكرر.
- **[النشر المستمر (CD)](./continuous-deployment/index.md):** ممارسة النشر التلقائي للتغييرات الناجحة في الإنتاج.
