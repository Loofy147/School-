# Key Concepts for SRE / DevOps / Observability
# المفاهيم الأساسية لـ SRE / DevOps / المراقبة

This section covers the core practices for building, deploying, and operating a reliable and efficient system.

- **[CI/CD (Continuous Integration/Continuous Deployment)](./ci-cd/index.md):** Automating the software delivery lifecycle.
- **[Infrastructure as Code (IaC)](./infrastructure-as-code/index.md):** Managing infrastructure through code.
- **[Observability](./observability/index.md):** Understanding the internal state of the system.
- **[SLIs, SLOs, and SLAs](./slis-slos-slas/index.md):** Defining and measuring reliability.
- **[Error Budgets](./error-budgets/index.md):** Balancing reliability with innovation.
- **[Runbooks](./runbooks/index.md):** Documenting operational procedures.

- **[مؤشرات مستوى الخدمة، وأهداف مستوى الخدمة، واتفاقيات مستوى الخدمة](./slis-slos-slas/index.md):** تحديد وقياس الموثوقية.
- **[ميزانيات الخطأ](./error-budgets/index.md):** الموازنة بين الموثوقية والابتكار.
- **[كتيبات التشغيل](./runbooks/index.md):** توثيق الإجراءات التشغيلية.
