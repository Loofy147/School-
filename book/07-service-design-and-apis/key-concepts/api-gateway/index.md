# API Gateway
# بوابة API

A single entry point for all clients. The API gateway handles requests by routing them to the appropriate backend service, and can also be used for cross-cutting concerns like authentication, rate limiting, and logging.

---

نقطة دخول واحدة لجميع العملاء. تتعامل بوابة API مع الطلبات عن طريق توجيهها إلى خدمة الواجهة الخلفية المناسبة، ويمكن استخدامها أيضًا للاهتمامات الشائعة مثل المصادقة وتحديد المعدل والتسجيل.
