# gRPC (Google Remote Procedure Call)
# gRPC (استدعاء الإجراء البعيد من Google)

A high-performance, open-source universal RPC framework. It uses Protocol Buffers as its interface definition language and is often used for high-throughput, low-latency communication between backend services. It operates over HTTP/2.
