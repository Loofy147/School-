# REST (نقل الحالة التمثيلية)

An architectural style for designing networked applications. It's based on a stateless, client-server communication model and the use of standard HTTP methods (GET, POST, PUT, DELETE). It typically uses JSON over HTTP/1.1.

---

نمط معماري لتصميم التطبيقات المتصلة بالشبكة. يعتمد على نموذج اتصال عميل-خادم عديم الحالة واستخدام أساليب HTTP القياسية (GET ، POST ، PUT ، DELETE).
