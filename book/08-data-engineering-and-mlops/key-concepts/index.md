# Key Concepts for Data Engineering & MLOps
# المفاهيم الأساسية لهندسة البيانات و MLOps

This section covers the core concepts for building the data pipelines and machine learning operations that power our intelligent system.

- **[ETL (Extract, Transform, Load)](./etl/index.md):** The fundamental process for moving and preparing data.
- **[MLOps (Machine Learning Operations)](./mlops/index.md):** Applying DevOps principles to the machine learning lifecycle.
- **[Data Quality](./data-quality/index.md):** Ensuring the accuracy and reliability of our data.

يغطي هذا القسم المفاهيم الأساسية لبناء خطوط أنابيب البيانات وعمليات تعلم الآلة التي تشغل نظامنا الذكي.

- **[ETL (استخراج، تحويل، تحميل)](./etl/index.md):** العملية الأساسية لنقل البيانات وإعدادها.
- **[MLOps (عمليات تعلم الآلة)](./mlops/index.md):** تطبيق مبادئ DevOps على دورة حياة تعلم الآلة.
- **[جودة البيانات](./data-quality/index.md):** ضمان دقة وموثوقية بياناتنا.
