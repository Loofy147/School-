# Tools for Data Engineering & MLOps
# أدوات لهندسة البيانات و MLOps

## Data Pipeline & ETL Tools / أدوات خط أنابيب البيانات و ETL

- **Apache Airflow:** A popular open-source platform for programmatically authoring, scheduling, and monitoring workflows.
- **dbt (data build tool):** A tool that allows you to transform data in your warehouse more effectively using SQL.
- **Spark / Flink:** Open-source distributed computing systems for large-scale data processing.
- **Matillion / Fivetran:** Commercial, cloud-based data integration platforms that provide a wide range of pre-built connectors.

- **Apache Airflow:** منصة مفتوحة المصدر شائعة لإنشاء وجدولة ومراقبة مهام سير العمل برمجيًا.
- **dbt (أداة بناء البيانات):** أداة تتيح لك تحويل البيانات في مستودعك بشكل أكثر فعالية باستخدام SQL.
- **Spark / Flink:** أنظمة حوسبة موزعة مفتوحة المصدر لمعالجة البيانات على نطاق واسع.
- **Matillion / Fivetran:** منصات تكامل بيانات تجارية قائمة على السحابة توفر مجموعة واسعة من الموصلات المعدة مسبقًا.

## MLOps Platforms & Tools / منصات وأدوات MLOps

- **MLflow:** An open-source platform for managing the ML lifecycle.
- **Kubeflow:** An open-source toolkit for ML on Kubernetes.
- **Weights & Biases / Comet:** Commercial platforms for experiment tracking and model management.
- **Feast:** An open-source feature store.

- **MLflow:** منصة مفتوحة المصدر لإدارة دورة حياة تعلم الآلة.
- **Kubeflow:** مجموعة أدوات مفتوحة المصدر لتعلم الآلة على Kubernetes.
- **Weights & Biases / Comet:** منصات تجارية لتتبع التجارب وإدارة النماذج.
- **Feast:** متجر ميزات مفتوح المصدر.

## Data Quality Tools / أدوات جودة البيانات

- **Great Expectations:** An open-source Python library for data validation, profiling, and documentation.
- **Soda Core:** An open-source framework for data reliability and quality engineering.
- **Monte Carlo / Bigeye:** Commercial data observability platforms that help you detect and diagnose data quality issues.

- **Great Expectations:** مكتبة Python مفتوحة المصدر للتحقق من صحة البيانات وتحديد مواصفاتها وتوثيقها.
- **Soda Core:** إطار عمل مفتوح المصدر لموثوقية البيانات وهندسة الجودة.
- **Monte Carlo / Bigeye:** منصات ملاحظة بيانات تجارية تساعدك في اكتشاف وتشخيص مشكلات جودة البيانات.
