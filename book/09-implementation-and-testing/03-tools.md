# Tools for Implementation & Testing
# أدوات للتنفيذ والاختبار

## Linters & Formatters / أدوات التدقيق والتنسيق

- **ESLint / Prettier (for JavaScript/TypeScript):** Tools that automatically find and fix problems in your code, as well as enforce a consistent code style.
- **Black / Flake8 (for Python):** Popular tools for formatting and linting Python code.
- **SonarLint:** A free and open-source IDE extension that identifies and helps you fix quality and security issues as you code.

- **ESLint / Prettier (لـ JavaScript / TypeScript):** أدوات تجد المشكلات في الكود الخاص بك وتصلحها تلقائيًا، بالإضافة إلى فرض نمط كود متسق.
- **Black / Flake8 (لـ Python):** أدوات شائعة لتنسيق وتدقيق كود Python.
- **SonarLint:** امتداد IDE مجاني ومفتوح المصدر يحدد ويساعدك على إصلاح مشكلات الجودة والأمان أثناء كتابة التعليمات البرمجية.

## Unit Testing Frameworks / أطر اختبار الوحدة

- **Jest / Vitest (for JavaScript/TypeScript):** Popular and easy-to-use testing frameworks for JavaScript.
- **Pytest (for Python):** A powerful and flexible testing framework for Python.
- **JUnit (for Java):** The standard testing framework for the Java programming language.

- **Jest / Vitest (لـ JavaScript / TypeScript):** أطر اختبار شائعة وسهلة الاستخدام لـ JavaScript.
- **Pytest (لـ Python):** إطار اختبار قوي ومرن لـ Python.
- **JUnit (لـ Java):** إطار الاختبار القياسي للغة برمجة Java.

## E2E Testing Frameworks / أطر اختبار E2E

- **Cypress:** A modern, all-in-one testing framework for web applications.
- **Playwright:** An open-source framework from Microsoft for reliable end-to-end testing for modern web apps.
- **Selenium:** A long-standing and widely used framework for automating web browsers.

- **Cypress:** إطار اختبار حديث ومتكامل لتطبيقات الويب.
- **Playwright:** إطار عمل مفتوح المصدر من Microsoft لاختبار شامل وموثوق لتطبيقات الويب الحديثة.
- **Selenium:** إطار عمل قديم ومستخدم على نطاق واسع لأتمتة متصفحات الويب.
