Templates package for Top-Down Systems Engineering course.
Files included:
- ADR_TEMPLATE.md
- c4_plantuml_example.puml
- RTM_TEMPLATE.csv
- PACT_QUICKSTART.md
- CAPSTONE_BRIEF.md

Usage:
- Copy templates into your course repo.
- Edit ADR_TEMPLATE.md for each architectural decision.
- Use c4_plantuml_example.puml with PlantUML to generate diagrams.
- Fill RTM_TEMPLATE.csv for traceability.
- Follow PACT_QUICKSTART.md to add contract tests and set up a Pact Broker.
