Templates package for Top-Down Systems Engineering course.
حزمة قوالب لمساق هندسة النظم من القمة إلى القاعدة.

Files included / الملفات المضمنة:
- ADR_TEMPLATE_bilingual.md (قالب ADR ثنائي اللغة)
- c4_plantuml_example_bilingual.puml (قالب C4 / PlantUML — مثال ثنائي اللغة)
- RTM_TEMPLATE_bilingual.csv (قالب مصفوفة التتبع — عناوين ثنائية اللغة)
- PACT_QUICKSTART_bilingual.md (دليل Pact ثنائي اللغة)
- CAPSTONE_BRIEF_bilingual.md (بيان مشروع التخرج ثنائي اللغة)

Usage / التعليمات:
- Copy templates into your course repo. / انسخ القوالب إلى مستودع المساق.
- Edit ADR_TEMPLATE_bilingual.md for each architectural decision. / عدّل ADR لكل قرار معماري.
- Use c4_plantuml_example_bilingual.puml with PlantUML to generate diagrams. / استخدم PlantUML لتوليد المخططات.
- Fill RTM_TEMPLATE_bilingual.csv for traceability. / عبئ مصفوفة التتبع.
- Follow PACT_QUICKSTART_bilingual.md to add contract tests and set up a Pact Broker. / اتبع الدليل لإضافة اختبارات العقود وإعداد Pact Broker.
